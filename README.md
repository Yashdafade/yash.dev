A developer portfolio built to look and feel exactly like Visual Studio Code. Navigate through my projects, skills, and experience as if you're browsing a codebase — complete with a file explorer sidebar, tabs, and syntax-highlighted content.

**Live:** [https://portfolio-v5-vs.vercel.app](https://portfolio-v5-vs.vercel.app)

## Concept

Instead of a conventional portfolio layout, this one mimics the VS Code IDE interface — because it's where I spend most of my time anyway. The file tree maps to portfolio sections: About, Projects, Skills, Contact.

## Tech Stack

- **React** — component architecture and routing
- **Vite** — fast dev build tooling
- **CSS** — custom VS Code theme replication
- **Vercel** — deployment

## Features

- VS Code-style file explorer sidebar
- Tab navigation between sections
- Syntax-highlighted content panels
- Status bar and activity bar replication
- Fully responsive

## Running Locally

```bash
git clone https://github.com/Yashdafade/yash.dev.git
cd yash.dev
npm install
npm run dev
```

Open http://localhost:5173

## Deployment

Deployed on Vercel. Any push to `main` triggers a new deployment automatically.
